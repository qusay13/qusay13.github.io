# كتاب زينب — هدية عيد ميلاد تفاعلية

موقع React (Vite) يعرض كتاباً تفاعلياً بغلاف مزخرف، يُفتح بحركة دوران ثلاثية الأبعاد،
ثم يمكن تقليب صفحاته (30 صفحة قصائد) واحدة تلو الأخرى مثل كتاب فعلي، وينتهي بصندوق رسالة شخصية.

## التشغيل

```bash
npm install
npm run dev
```

افتح الرابط الذي يظهر في الطرفية (عادة http://localhost:5173).

## البناء للنشر

```bash
npm run build
```

الناتج يكون داخل مجلد `dist/` — يمكنك رفعه على أي استضافة ثابتة
(Vercel, Netlify, GitHub Pages, Cloudflare Pages...) لأنه frontend بالكامل بدون باك إند.

## أين أعدّل المحتوى؟

- **القصائد الثلاثون**: ملف `src/data/poems.js`
  كل قصيدة عبارة عن `{ title: "العنوان", lines: ["سطر 1", "سطر 2", ...] }`.
  بدّل أي نص كما تريد، فقط حافظ على بقاء العدد الإجمالي 30 (أو رقم زوجي إن غيّرت العدد،
  لأن كل ورقة فعلية تحمل قصيدتين: واحدة بالواجهة وواحدة بالظهر).

- **الرسالة الشخصية بآخر الكتاب**: ملف `src/components/MessagePage.jsx`،
  المتغيّر `messageText` بالأعلى — عدّله بجملك الخاصة.

- **اسم الغلاف "Zainab" والعنوان الفرعي**: ملف `src/components/BookCover.jsx`.

- **الألوان**: كل الألوان معرّفة كمتغيرات CSS في `src/index.css` تحت `:root`
  (`--rose-deep`, `--rose-mid`, `--rose-pale`, `--blush-bg`, `--gold`...) — غيّرها من مكان واحد
  لتنعكس بكل الموقع.

## ملاحظات تقنية

- المشروع React عادي بـ Vite، بدون أي باك إند أو API خارجي.
- تقليب الصفحات مبني بـ CSS 3D transforms يدوياً (rotateY + perspective) بدون مكتبات خارجية،
  لضمان أداء خفيف وسلس على الموبايل.
- يدعم السحب باللمس (swipe) على الموبايل للتنقل بين الصفحات، بالإضافة لأزرار "السابقة/التالية".
- يحترم `prefers-reduced-motion` لمن يفضّل تقليل الحركة.
